# Dataset Access Policy

Access to this dataset is **not automatic**.

---

## Request Process

Submit a request via:
- GitHub Issue labeled `dataset-access-request`, or
- Email to **data@hollowhouse.org**

---

## Evaluation Criteria
- Consent alignment
- Governance purpose
- Non-extractive intent
- Duration & scope

---

## Approved Access Includes
- Personalized license
- Secure delivery instructions
- Versioned documentation
