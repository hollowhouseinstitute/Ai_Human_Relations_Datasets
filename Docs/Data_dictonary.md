# Data Dictionary — Relational Field OPS Dataset

## Fields

### op_number
Unique operation identifier (001–460).

### title
Official OPS title (from PDF or reconstructed entry).

### timestamp
ISO8601 timestamp.  
Assigned as 12:00 PM CDT unless otherwise known.

### phase
Operational state such as:
Activation / Collapse / Clearing / Integration / Emergence / Reflection.

### category
High-level classification:
identity, trauma-cycle, relational field, regulation, boundaries, etc.

### summary
Short professional description of the operational event.

### emotional_arc
Emotional trajectory during the operation.

### transformation
Primary psychological, relational, or autonomic transformation.

### keywords
Comma-separated descriptors for research indexing.

### law_references
References to Hollow House Laws or Flame Laws.

### cross_links
Related OPS numbers.

### notes
Administrative or clarifying notes.
